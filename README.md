# Actividad README en GitHub

## Descripción

Este proyecto tiene como finalidad mostrar cómo crear un repositorio en GitHub utilizando un archivo README escrito en Markdown.

La práctica incluye instrucciones de instalación, requisitos del sistema, ejemplo de uso y una captura del resultado esperado.

---

## Requisitos

Para utilizar este proyecto se necesita:

- Git instalado
- Cuenta de GitHub
- Visual Studio Code
- Navegador Web

---

## Instalación

1. Abrir una terminal.
2. Clonar el repositorio.

```bash
git clone https://github.com/TUUSUARIO/Actividad-README.git
```

3. Entrar a la carpeta.

```bash
cd Actividad-README
```

4. Abrir el proyecto en Visual Studio Code.

```bash
code .
```

---

## Ejemplo de uso

Guardar los cambios.

```bash
git add .
```

Crear un commit.

```bash
git commit -m "Primer commit"
```

Subir los cambios.

```bash
git push origin main
```

---

## Resultado esperado

Después de ejecutar los comandos, el proyecto deberá aparecer correctamente en GitHub con todos los archivos actualizados.
